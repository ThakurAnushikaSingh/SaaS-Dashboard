# TaskFlow SaaS Dashboard

A modern SaaS dashboard prototype for a project management product.

## Features
- Login screen
- Dashboard overview
- Project management
- Team management
- Analytics
- Settings
- Dark mode toggle
- Clickable navigation and notifications

## Tools Used
- HTML
- CSS
- JavaScript
- Figma-style layout system
- Auto Layout-inspired components
- Smart Animate-inspired transitions

## Prototype Link
Open `index.html` in this folder, or run a local static server from the project folder.
